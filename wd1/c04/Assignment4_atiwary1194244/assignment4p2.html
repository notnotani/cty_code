Some Web Design Terminology


Browser
A web browser is software that enables you to view and interact with web pages.
ISP
Internet Service Provider
An ISP is a company that provides you with connection to the Internet.
URL
Uniform Resource Locator
A URL, or Uniform Resource Locator is a unique address for a web page. For example, http://www.example.com/index.html.
Web Browser
See Browser
Worldwide Web
A Worldwide Web is a giant collection of documents, called web pages, stored on computers all over the world.
